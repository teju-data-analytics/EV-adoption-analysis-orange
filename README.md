# Electric Vehicle Adoption Analysis

Author: Maria Tejaswini Polishetty  
Course: Business Intelligence Data Analysis  

## Project Overview

This project analyzes electric vehicle (EV) adoption trends using a dataset of EV registrations in Washington State. The goal is to understand factors influencing CAFV (Clean Alternative Fuel Vehicle) eligibility and identify patterns in EV adoption.

Machine learning techniques were applied to analyze relationships between vehicle attributes such as electric range, MSRP, and vehicle type.

---

## Dataset

The dataset contains information about 1000 electric vehicles including:

- County and City
- Vehicle Make and Model
- Model Year
- Electric Range
- MSRP (Vehicle Price)
- Electric Vehicle Type
- CAFV Eligibility
- Geographic location data

---

## Methods Used

The following machine learning algorithms were used in the analysis:

### Logistic Regression
Used to classify CAFV eligibility based on vehicle attributes.

### Decision Tree
Used to analyze feature importance and decision paths.

---

## Key Insights

- Electric vehicles with **range greater than 100 miles** are more likely to qualify for CAFV eligibility.
- EV adoption increased significantly after **2015**.
- **Tesla Model 3 and Nissan Leaf** dominate EV registrations.
- Urban counties show higher EV adoption rates.
